## Communication

Meldog local network has access to the ethernet via wlan interface currently working on the 2,4 GHz bandwitch.

Local wlan network is being distributed via 5 GHz bandwith interface.

5 GHz wlan interface is bridged with a physical interface, thanks to this devices connected by a network cable can communicate with devices connected by a WLAN.

## Setup

Current network setup has its disadvantages:
1. Not every devices working in local wlan network accept 5 GHz bandwith. 
2. To change the network from which Meldog's network 'takes' the Inthernet, you need to manually set it in router network configuration.
